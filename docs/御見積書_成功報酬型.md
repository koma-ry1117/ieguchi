## メール機能の詳細

### 実装されるメール機能

| # | 機能名 | 詳細 |
|---|---|---|
| 1 | **内見予約確定メール** | 顧客・オペレーター両方に送信<br>カレンダー招待(.ics)自動添付<br>Google/Outlook等のカレンダーに直接追加可能 |
| 2 | **問い合わせ受付確認メール** | 問い合わせフォーム送信時の自動返信 |
| 3 | **システムお知らせメール** | メンテナンス通知、新機能案内等 |
| 4 | **パスワードリセットメール** | Supabase Auth経由での認証連携 |
| 5 | **送信ログ管理** | 監査用のメール送信履歴記録 |

### カレンダー招待(.ics)機能

- 内見予約時に自動生成
- 予約日時、物件情報、担当者情報を含む
- 主要カレンダーアプリ対応(Google Calendar、Outlook、Apple Calendar等)
- リマインダー設定(24時間前、1時間前)

### メール送信基盤

- Resend API使用(信頼性の高い配信)
- HTMLメールテンプレート対応
- 送信失敗時のリトライ機能
- 配信状況トラッキング
