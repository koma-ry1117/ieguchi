## コスト最適化戦略

### 1. 音声キャッシング（ElevenLabs削減）

**手法**: よくある応答を事前生成してSupabase Storageに保存

```typescript
const commonPhrases = [
  "かしこまりました。",
  "他にご希望の条件はございますか？",
  "以下の物件がございます。",
];

// 事前生成・キャッシュ
for (const phrase of commonPhrases) {
  const audio = await elevenlabs.textToSpeech.convert(...);
  await supabase.storage.upload(`cache/${hash(phrase)}.mp3`, audio);
}
```

**想定削減**: ¥10,000〜¥12,000/月（¥14,850 → ¥1,650 Creatorプラン）

### 2. 会話キャッシング（OpenAI削減）

**手法**: 同じ質問への回答をSupabaseにキャッシュして再利用

```typescript
const cachedResponse = await supabase
  .from('response_cache')
  .select('response')
  .eq('query_hash', hash(userQuery))
  .single();

if (cachedResponse) return cachedResponse.response;
```

**想定削減**: 30%（¥1,500〜¥2,000/月）

### 3. GPT-4o-mini の活用

**手法**: 簡単な質問には GPT-4o-mini を使用（コスト1/10）

**想定削減**: 40%（¥2,000〜¥2,500/月）
